ПРАВИЛА БЕЗОПАСНОСТИ И ОСОБЕННОСТИ ПОДПИСАНИЯ

КАРТЫ С КОДАМИ:
- Выдаются банком в момент получения ЭЦП
- Обязательны для владельцев ЭЦП с правом второй подписи
- Содержат ограниченное количество кодов
- Код запрашивается произвольно, не в каждом сеансе
- Один код используется только один раз за сеанс
- При невводе кода операция прерывается

КАРТА С ДИСПЛЕЕМ (ГЕНЕРАТОР КОДОВ):
- Альтернатива карте с кодами
- Генерирует коды динамически
- При повреждении можно использовать резервную карту доступа
- Резервную карту получают в офисе банка

СМС-ПОДТВЕРЖДЕНИЕ:
- Для низкорисковых операций
- SMS отправляется на номер с "Базовым" пакетом информирования
- Документ можно отредактировать до отправки
- Статусы документа: "Готовится", "Отбракован", "Подписан"

ВАЖНО: Храните карты и генераторы кодов в безопасном месте, не передавайте третьим лицам.
